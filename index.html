About the game

This game is a ripoff of the legendary webgame BMTron, which was later remade for HTML5 as Serpentron

How To Play

  • Add 2 or more players
  • The C button on each player card allows you to configure contols (saved in cookies for later)
  • Click "Configure Parameters" below the canvas to change the parameters of the game
  • Have Fun

Game Modes and Settings

  • Sprint Mode - Hold down sprint button to move twice as quick. Sprint recharges over time and is shown in your player card
  • Sprint Gaps - Every other block whilst sprinting left blank
  • Bombs - Press your bomb button when your bomb guage (in the player card) is full to remove tails
  • Wrap - Players can wrap around the map
  • BR Mode - Battle Royale mode, because that's all the rage these days
  • No Tail Zones - Randomly spawning zones where tails don't stay. Can be destroyed by bombs
  • Fast Zones - Randomly spawning zones where you move quicker. Can be destroyed by bombs
  • Grid Size - Change the grid size to vary play styles. Choose grid soze of 10-200
  • Game Speed - Choose an FPS of 10-60
